Welcome to C R U D – X

🌐 A web application that helps you manage your tasks 📜

    📝 You can Create a task and set
    • Duration ⏳,
    • Deadlines 📆,
    • Priority 🟡
    👀 Simply view it in Read section
    ⬆️ In Update section, you can
    • modify details of any task's by changing the Title, Duration, Duedate, Priority, etc...
    • Set a task as done by modifying its ✅ Status
    🗑️ Finally, Delete the task in its section when you no longer want it in the list

➕ Additions features to be added are:

  • 🔍 Search tasks by title.
  • 📊 Progress stats of task done/not done in the current list.
  • 📋 Sorting and filtering on due/deadline/priority
  • ☑️ Select multiple tasks to delete at once
